# Form Validator

Custom form validation

Check the demo hosted on https://filipeqs.github.io/form-validator/

---

## Instructions

First clone this repository

```bash
$ git clone https://github.com/filipeqs/form-validator.git
```

Open index.html

```bash
open index.html
```

## Features

- Use of HTML form validation
- Validate Password
